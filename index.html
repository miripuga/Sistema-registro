1 - Para verificar los cambios realizados: Utiliza el comando git status, sirve para listar todos los archivos que han sido modificados. 2 - Para agregar los cambios al repositorio local: Para agregar los cambios realizados de una sola vez, es necesario usar git add . (git add y un punto) y, para agregar los cambios en algún archivo específico, usamos a git add nombre-del-archivo-modificado. 3 - Para guardar los cambios: Utiliza el comando git commit, que sirve para capturar y guardar el estado actual del repositorio. 4 - Para enviar las modificaciones al repositorio remoto: Utiliza el comando git push, que sirve para enviar las modificaciones guardadas en el directorio local para el repositorio remoto.
